# Articolare User Access API

```
This is a RESTful API for the Articolare application. 

The application follows basic coding principles and best practices. This RESTful API follows oAUTH 2 protocols. 
```

## Project setup

```
In order to run the programme, please download all the necessary packages listed in the commnds.md file.

* Packages needed for this application 

    npm install express nodemon mongoose dotenv body-parser axios

* Package used for validation

    npm install joi

* Packages used for emcripting passwords

    npm install bcryptjs

* Packages used for auth-token 

    npm install jsonwebtoken
